API Map

GET /api/v1/stores/
Возвращает список магазинов
GET /api/v1/stores/<store_id>/
Возвращает информацию о конкретном магазине и список всех товаров в наличии на момент запроса с их количеством
POST /api/v1/stores/<store_id>/buy/
Ендпоинт для покупки товаров в магазине
POST /api/v1/stores/<store_id>/add/
Ендпоинт для добавления товара в магазин.
